
# DrugsMod (Unity 2019.4.10f1 LTS)

A modification of mines3 built with Unity 2019.4.10f1 LTS.

## Requirements

- Unity 2019.4.10f1 (LTS)

## How to Build and Run

### 1. Clone the Repository

git clone https://github.com/Touch-the-rosess/DrugsMod.git

cd DrugsMod

### 2. Open the Project in Unity

1. Open Unity Hub.
2. Click "Add".
3. Select the project root folder.
4. Open the project using Unity 2019.4.10f1 (LTS).

### 3. Run in Editor

1. Open the scene: Assets/Scenes/m1client.unity
2. Press Play.

### 5. Build the Project

1. Go to File â†’ Build Settings.
2. Ensure the following scenes are in "Scenes In Build":
   - Assets/Scenes/m1client.unity
3. Click "Build".
4. Choose an empty folder.
5. Wait for the build to complete.
